# Image Viewer Simple

Простое приложение на C# Windows Forms для открытия и просмотра изображений.

## Функциональность

- Кнопка для выбора изображения
- OpenFileDialog с фильтром по типам файлов
- Поддержка JPG, JPEG, PNG, GIF, BMP
- Отображение в PictureBox
- Обработка ошибок загрузки

## Интерфейс

- btnOpenImage — кнопка открытия
- pictureBox1 — отображение картинки

## Куда можно встроить

- В просмотрщики изображений
- В программы для работы с графикой
- В демонстрацию работы OpenFileDialog

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При повторном открытии предыдущее изображение заменяется.
